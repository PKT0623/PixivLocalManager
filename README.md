# Pixiv Local Manager

로컬에 저장된 Pixiv 작품 폴더를 관리하기 위한 개인용 데스크톱 프로그램.

작가 등록, 작품 수 관리, 업데이트 확인, 백업 및 복구 기능을 제공하며, 장기적으로는 작품 관리 시스템과 자체 뷰어 기능까지 포함하는 것을 목표로 한다.

---

# 프로젝트 개요

Pixiv 작품을 장기간 수집하다 보면 작가 수와 작품 수가 많아져 관리가 어려워진다.

Pixiv Local Manager는 다음 문제를 해결하기 위해 개발되었다.

* 저장된 작가 목록 관리
* 로컬 작품 보유 현황 관리
* 업데이트 필요 작가 확인
* 누락 작품 확인
* 작가별 메모 및 태그 관리
* 데이터 백업 및 복구
* 대규모 로컬 아카이브 관리

# 실행 방법

## 가상환경 생성

```bash
python -m venv .venv
```

## 가상환경 활성화

```bash
.venv\Scripts\activate
```

## 패키지 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
python main.py
```
