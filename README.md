# Pixiv Local Manager

로컬에 저장된 Pixiv 작품 폴더를 체계적으로 관리하기 위한 개인용 데스크톱 프로그램.

작가 관리, 폴더 스캔, 업데이트 확인, 대시보드, 백업 및 복구 기능을 제공하며, 장기적으로는 작품 관리 시스템과 자체 뷰어 기능까지 포함하는 것을 목표로 한다.

---

# 프로젝트 소개

Pixiv 작품을 장기간 수집하다 보면 작가 수와 작품 수가 증가하면서 다음과 같은 문제가 발생한다.

* 어떤 작가가 업데이트되었는지 확인하기 어렵다.
* 어떤 작품이 누락되었는지 확인하기 어렵다.
* 다운로드 기준이나 메모를 관리하기 어렵다.
* 작가별 작품 규모를 파악하기 어렵다.
* 장기간 보관 시 데이터 관리가 번거롭다.

Pixiv Local Manager는 이러한 문제를 해결하기 위해 개발되었다.

# 설치 방법

## 저장소 복제

```bash
git clone https://github.com/PKT0623/PixivLocalManager.git
cd PixivLocalManager
```

---

## 가상환경 생성

```bash
python -m venv .venv
```

---

## 가상환경 활성화

Windows PowerShell

```powershell
.venv\Scripts\activate
```

---

## 패키지 설치

```bash
pip install -r requirements.txt
```

---

## 프로그램 실행

```bash
python main.py
```

# 최종 목표

Pixiv 작품을 장기간 수집하는 사용자가

```text
작가 관리
→ 업데이트 확인
→ 통계 및 분석
→ 작품 관리
→ 작품 감상
```

까지 하나의 프로그램 안에서 수행할 수 있는 통합 관리 환경을 제공하는 것을 목표로 한다.
