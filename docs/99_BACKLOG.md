# Backlog

## 설명

현재 V1 범위에 포함되지 않았거나, V2 이후 추가 개발을 고려하는 기능 목록.

완료된 기능은 하단의 `구현 완료 항목`으로 이동하여 관리한다.

우선순위는 존재하지 않으며, 기능 추가 시 카테고리 단위로 진행한다.
