# Backlog

## 설명

현재 V1 범위에 포함되지 않았지만 추후 개발을 고려하는 기능 목록.

우선순위는 존재하지 않으며, 기능 추가 시 카테고리 단위로 진행한다.

아이디어 보관 및 향후 개발 계획 수립 용도로 사용한다.
